## Phase 3.5 QA Checklist

执行环境：本地 mock 模式（`ANALYZER_PROVIDER=mock`），Chrome 最新版。

执行方式：
```bash
# 后端
cd backend && source .venv/bin/activate
uvicorn app.main:app --reload

# 前端（另开终端）
cd frontend && npm run dev
# 访问 http://localhost:5173
```
